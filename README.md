
## Project Description

This is a meat store listing page. The page displays each product item, and users can delete or edit products. In addition, users can also create new products.

On the product listing page, users are able to group products by category ID or unit of measure (UOM), and also sort the product list.

## Tech Stack

- **Next.js** – React framework
- **Material UI (MUI)** – UI components
- **Redux Toolkit** – State management
- **TypeScript** – Type safety

## Local Development Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Open your browser:**
   Go to [http://localhost:3000](http://localhost:3000)

Your Next.js app will be running locally with all features enabled.
